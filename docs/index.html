Install

Up in 30 seconds

Choose your install method. Works with every major framework.

TERMINAL
# Install the package
npm install @salkomdesignstudio/sds-motion-forge

# Then import in your entry file
import '@salkomdesignstudio/sds-motion-forge/dist/motion.min.css';
HTML
<!-- Apply with a single class -->
<h1 class="sds-velvet-drop">Hello World</h1>

<button class="sds-btn-magnetic">Click Me</button>

<div class="sds-card-float">Card</div>

<!-- Stack modifiers -->
<h2 class="sds-liquid-rise sds-slow sds-delay-3">
  Delayed & Slow
</h2>

Utilities

Modifier classes

How It Works

Three steps, zero friction

1
Install or link

Drop in the CSS via npm or CDN — no build step required.

npm i @salkomdesignstudio/sds-motion-forge
2
Add a class

Apply an animation class to any HTML element.

<h1 class="sds-velvet-drop">
3
Stack modifiers

Combine speed, delay, and loop utilities for full control.

class="sds-velvet-drop
 sds-slow sds-delay-3"
