Why Motion Forge

Engineered for production

Not a toy. A focused, opinionated motion system built for teams who want character and precision without the overhead.

Zero Required JS
Pure @keyframes and CSS classes. No required JavaScript. Two optional engines (scroll gate, interactive) add progressive enhancement when you need it.
300+
Animations
Across six categories. 100 text alone.
Framework-agnostic
Angular, React, Vue, Next, Flutter, Bootstrap, Tailwind — anywhere a class works.
Themeable
Override two CSS variables and the whole library matches your brand.
GPU-accelerated
Transform & opacity only. Buttery on mobile, no layout thrash.
Accessible
Respects prefers-reduced-motion out of the box. No config needed.
