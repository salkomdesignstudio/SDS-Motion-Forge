# Code of Conduct

This project adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## The short version

Be kind, be honest, be patient. Critique designs and code, never people.
Harassment, discrimination, trolling, doxxing, and sustained disruption are
not tolerated anywhere this project lives — issues, PRs, discussions, or
community spaces referencing the project.

## Enforcement

Report conduct issues to **hello@salkomdesignstudio.com**. Reports are
reviewed by the maintainers, kept confidential, and answered within 7 days.
Consequences scale from a private warning to a permanent ban, at the
maintainers' judgment, per the Contributor Covenant enforcement guidelines.
